# 🔔 Remindly - 네이버 예약 보완용 노쇼(No-Show) 방지 및 고객 알림 레이어 서비스

> **"네이버 예약의 아쉬움을 채우는 사장님들의 똑똑한 예약 파트너, Remindly"**
>
> Remindly는 예약 취소 및 노쇼 예방을 돕고, 위약금 관련 불필요한 고객 분쟁과 감정 소모를 줄여주는 네이버 예약 보완용 관리 대시보드 및 모바일 포털 MVP 서비스입니다.

## 💡 기획 의도 (Planning Intent)

* **네이버 예약의 관리 공백 보완**: 기존 네이버 예약 시스템은 사장님들에게 훌륭한 창구이지만, **노쇼(No-Show) 방지 및 패널티/위약금 청구와 고객 소통**에 있어서는 별도의 리소스와 수동 처리가 요구되는 불편함이 있었습니다.
* **고객 리마인드를 통한 노쇼 차단**: 예약 시간 직전 고객에게 알림톡(메시지)을 전송하고, 단 한 번의 클릭만으로 방문 여부 확인 및 시간 변경 등을 고객이 스스로 선택할 수 있도록 설계하여 노쇼율을 획기적으로 낮춥니다.
* **약관 기반의 합리적인 정산서 제공**: 예약 취소 시 위약금/환불액 계산 분쟁을 방지하기 위해 등록된 약관에 입각한 정식 **"위약금 정산서"** 문구를 즉시 빌드 및 발송할 수 있도록 돕습니다.
* **감정 노동의 최소화**: 톤앤매너(친절하게, 단호하게, 고급스럽게)가 가미된 상황별 알림 문구를 빠르게 복사해 보낼 수 있어, 바쁜 업장 운영 중 문자 작성에 드는 시간과 스트레스를 덜어줍니다.

## ✨ 핵심 기능 (Key Features)

### 1. 📊 사장님 대시보드 (Admin View)
* **종합 스코어보드**: 오늘 총 예약수, 확인 완료 고객, 확인 대기 중인 고객, 노쇼/취소 경고를 시각적인 카드로 파악합니다.
* **오늘의 리마인드 우선순위**: 확인 대기 상태인 고객(Pending)을 별도로 리스팅하여 사장님이 우선적으로 알림톡을 발송할 수 있게 목록화합니다.
* **예약 내역 관리**: 
  - 신규 예약을 수동으로 바로 등록할 수 있는 모달 폼을 제공합니다.
  - 예약 리스트를 검색(고객 이름, 전화번호)하고 상태별(확인 대기, 예약 확정, 예약 변경 요청, 노쇼 등)로 필터링합니다.

### 2. ⚙️ 업종별 최적화 정책 설정 & 카카오 알림톡 실시간 미리보기
* **업종별 원클릭 프리셋**: **파인다이닝, 미용실/헤어, 네일숍, 사진스튜디오, 병원/클리닉**의 평균적인 예약 정책(예약금, 노쇼 취소 기준, 지각 취소 시간)을 클릭 한 번으로 자동 세팅합니다.
* **유연한 정책 커스터마이징**: 매장명, 네이버 플레이스 주소, 예약금 수납 계좌 정보, 상세 취소/환불 규정을 업장에 맞게 설정할 수 있습니다.
* **메시지 전송 단계별 & 톤앤매너별 템플릿**:
  - **전송 단계**: 예약 확정, 1일 전 알림, 당일 알림, 방문 후 감사 메시지
  - **메시지 톤앤매너**: 친절하게 😊, 단호하게 ⚠️, 고급스럽게 ✨
  - 설정된 문구를 모바일 화면 형태의 **카카오 알림톡 템플릿 실시간 미리보기**로 확인하고 `미리보기 문구 복사하기`를 통해 빠르게 발송에 활용할 수 있습니다.

### 3. 🧮 위약금 및 환불 금액 정산서 빌더 (Estimator)
* 예약금과 취소 접수 시점(예약 2시간 이내, 24시간 이내, 48시간 이내, 무단 불참)을 설정하면, 설정된 위약금 부과 비율(%)에 따라 공제 금액과 실제 최종 환불/청구 금액을 소수점 및 수치 계산을 통해 자동으로 정산합니다.
* 계산 후 클릭 시 고객에게 바로 전송할 수 있는 **정형화된 고지 텍스트 정산서**를 자동으로 빌드해 줍니다.

### 4. 📱 고객 모바일 확인 포털 (Customer View)
* 사장님이 보낸 알림 문구의 링크를 클릭한 고객이 진입하는 화면입니다.
* **방문 확정**: 모바일 카드에서 `네, 정시 방문합니다` 버튼을 클릭하면 사장님의 대시보드 상태가 실시간으로 '확인 완료(Confirmed)'로 업데이트됩니다.
* **일정 변경 요청**: 예약 가능 시간 내에서 일정 변경 희망 일시와 사유를 입력하여 사장님께 상태를 전달(Change Requested)할 수 있습니다.
* **예약 취소**: 취소 사유를 적고 바로 예약을 취소할 수 있으며(Cancelled), 사장님이 지정한 환불 및 취소 규정이 모바일 페이지 하단에 자동으로 고지되어 불필요한 추가 확인 전화를 생략하게 해줍니다.

## 🛠️ 사용 방법 (Usage Instructions)

1. **앱 실행**: 상단의 웹 실행 링크를 클릭하거나 폴더 내 `index.html` 파일을 더블 클릭하여 웹 브라우저에서 엽니다.
2. **초기 세팅**:
   * [정책 & 메시지 설정] 탭으로 이동하여 업종 프리셋 버튼을 클릭하거나 업장의 정보를 기입하고 `정책 및 매장정보 저장`을 누릅니다.
3. **예약 등록 및 알림 전송**:
   * [예약 관리] 혹은 [대시보드]에서 `예약 신규 등록`을 눌러 고객 예약을 임의 생성합니다.
   * 리스트에서 `리마인드 복사` 버튼을 눌러 생성된 카카오 알림톡 템플릿을 복사하고 실시간 문자를 발송합니다.
4. **고객 피드백 시뮬레이션**:
   * 사장님 관리 대시보드에서 각 예약 우측의 **[고객 확인 시뮬레이션]** 버튼을 클릭하면, 해당 고객 전용 모바일 웹뷰 포털 페이지로 연결됩니다.
   * 여기서 고객 입장이 되어 `방문 확인`, `일정 변경 요청`, `예약 취소`를 진행해 보세요.
   * 다시 주소창에 `#dashboard`를 기입하거나 브라우저 뒤로가기를 실행하면, 고객의 응답에 따라 대시보드 현황이 실시간으로 업데이트된 모습을 확인할 수 있습니다.
5. **위약금 정산**:
   * 취소 분쟁 발생 시 [위약금 정산서] 탭으로 이동하여 정보를 입력한 뒤 계산하고, 생성된 정산 문자 텍스트를 복사하여 고객에게 안내합니다.
